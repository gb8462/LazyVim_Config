## Installation

Install neovim
```
sudo pacman -S neovim
```
Create a neovim config
```
mkdir ~/.config/nvim
```
Clone the repository
```
git clone https://github.com/gb8462/LazyVim_Config/ ~/.config/nvim
```
Open neovim
```
nvim
```
